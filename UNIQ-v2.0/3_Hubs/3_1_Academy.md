# 3.1 The Academy (Education Hub)

Entry point hub for all users.

## Features
- Flashcards
- Quizzes
- Missions & mini-challenges

## Rewards
- Dust: latent token reward
- XP: non-transferable merit points

## Example Workflow
Cadet completes 20 quizzes → earns 500 XP → eligible for Master Gate
